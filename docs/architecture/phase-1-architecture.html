Safety Level Behavior

Level Meaning Phase 1 action
L0 Normal route progress low/medium recording, checkpoint capsule sealing
L1 Uncertainty rises increase recording, retain richer summaries
L2 Concern open incident, lock raw window, record map evidence, recommend go/no-go
L3 Distress enable ack/reack and nearby-pull mock
L4 Emergency mock remote alert, preserve evidence chain

Phase 1 keeps safety decisions deterministic. Offline map evidence is the highest-priority static terrain reference, while GPS/PDR/IMU are noisy measurements against that context. The current baseline now supports both fixture replay and live SensorLog ingestion through the existing FastAPI app: normal observations remain L0, abnormal streams create L2 incidents, policy-based raw windows are preserved, and the legacy PDR update flow stays intact.
